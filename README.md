# SG-16 SIGNAL

Browser-based hardware-inspired sampler/groovebox. Pure HTML/CSS/JavaScript with Web Audio API—no plugins, no server, no dependencies.

## Getting Started

1. Clone this repo: `git clone https://github.com/garridocchristian-alt/sg16-groovebox.git`
2. Open `index.html` in your web browser
3. Drag audio files onto the pads, sequence beats, mix, and export

## Features

- **16-pad sampler** with waveform editor & transient detection
- **Step sequencer** (drum patterns, SP-1200/TR-808 inspired)
- **Polyphonic synth sequencer**
- **7-channel mixer** with 3-band EQ
- **Filter & modulation** (vintage modes, ADSR)
- **Effects rack** (reverb, bit-depth, saturation)
- **Session persistence** (localStorage auto-save)

[Full feature breakdown](https://github.com/garridocchristian-alt/sg16-groovebox/blob/main/FEATURES.md)

## Built With

- Web Audio API
- HTML5 Canvas
- Vanilla JavaScript (no frameworks)
